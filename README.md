# PaletteBypass
PaletteBypass is a server-side Fabric mod that allows server owners to remove and add content-adding mods without
dropping any blocks or items.

## Progress
- [x] Blocks
- [x] Items
- [x] Block Entities
- [x] Unloadable Chunks
- [x] Entities
- [ ] Commands to check unknown blocks/entities/items/chunks
